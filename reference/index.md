# Package index

## Data Retrieval

Functions to download data from government sources

- [`getFRED()`](https://mtkonczal.github.io/tidyusmacro/reference/getFRED.md)
  : Download and Merge FRED Series
- [`getBLSFiles()`](https://mtkonczal.github.io/tidyusmacro/reference/getBLSFiles.md)
  : Download and Process Bureau of Labor Statistics Data
- [`getNIPAFiles()`](https://mtkonczal.github.io/tidyusmacro/reference/getNIPAFiles.md)
  : Download and Process BEA NIPA Files with Fast Row Expansion
- [`getPCEInflation()`](https://mtkonczal.github.io/tidyusmacro/reference/getPCEInflation.md)
  : Load and Process Personal Consumption Expenditures (PCE) Inflation
  Data
- [`getUnrateFRED()`](https://mtkonczal.github.io/tidyusmacro/reference/getUnrateFRED.md)
  : Get Full Unemployment Rate from FRED
